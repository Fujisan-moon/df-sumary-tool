# df-summary-tool

This project provides a function that automatically extracts basic information from a given DataFrame.

## Execution Environment
- Jupyter Notebook or Python

## Processing Details
- When a DataFrame is provided, a function (MLBasicInfo) returns the following information:
  - The first 5 rows
  - The number of rows and columns
  - Data types
  - Number of data
  - Mean, median, and quartiles
  - Number of null values
- In a Jupyter environment, the first 5 rows, data types, number of data, mean, median, quartiles, and null counts are displayed in HTML format.



# df-summary-tool

Dieses Projekt stellt eine Funktion bereit, die automatisch grundlegende Informationen aus einem gegebenen DataFrame extrahiert.

## Ausführungsumgebung
- Jupyter Notebook oder Python

## Verarbeitungsinhalt
- Wird ein DataFrame übergeben, gibt eine Funktion (MLBasicInfo) folgende Informationen zurück:
  - Die ersten 5 Zeilen
  - Anzahl der Zeilen und Spalten
  - Datentypen
  - Anzahl der Datenpunkte
  - Mittelwert, Median und Quartile
  - Anzahl der Nullwerte
- In einer Jupyter-Umgebung werden die ersten 5 Zeilen, die Datentypen, die Anzahl der Datenpunkte sowie Mittelwert, Median, Quartile und die Anzahl der Nullwerte im HTML-Format angezeigt.
